## JPR Global Export Demo Site

This is a modern, single-page marketing website for **JPR Global Export**, inspired by the existing site at [`https://globalvictus.com/`](https://globalvictus.com/), but rebuilt with a clean, responsive design and without any spam content.

### Tech Stack

- **HTML5**: Semantic single-page layout (`index.html`)
- **CSS3**: Custom responsive styling (`styles.css`)
- **Vanilla JS**: Mobile navigation toggle (`script.js`)

### Structure

- **Hero**: Brand intro, key stats, and primary calls-to-action.
- **Products**: Cards for paper cups, bagasse tableware, areca tableware, and wooden cutlery.
- **About**: Company story and values.
- **Associations**: Placeholder badges for ISO, GST, MSME, and B2B portals.
- **Catalogue / Social**: Links to request a catalogue and connect on social platforms.
- **Contact**: Address, phone, email, hours, and a non-functional quote form.
- **Footer**: Quick links, contact info, and social links.

### How to View Locally

1. Open the project folder:
   ```bash
   cd "/Users/mkansodiya/Desktop/Projects/TupperwareBiz"
   ```
2. Open `index.html` directly in a browser:
   - On macOS Finder: double-click `index.html`, or
   - From terminal:
     ```bash
     open index.html
     ```
3. (Optional) Use a local web server (for best experience):
   ```bash
   python3 -m http.server 8000
   ```
